## Build from source

1. Clone the repo
   ```sh
   https://github.com/dash4k/informatika-semester-3.git
   ```
2. Go to directory folder
   ```sh
   cd /informatika-semester-3/tbo/tugas-kelompok-2
   ```
3. Execute the main file
   ```sh
   python main.py
   ```
