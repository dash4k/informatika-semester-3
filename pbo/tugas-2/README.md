## Build from source

1. Clone the repo
   ```sh
   https://github.com/dash4k/informatika-semester-3.git
   ```
2. Go to the build directory
   ```sh
   cd /pbo/tugas-2
   ```
3. Make the binary
   ```sh
   javac -d bin src/Laptop.java src/Shop.java src/Main.java
   ```
4. Run the binary
   ```sh
   java -cp bin Main
   ```
